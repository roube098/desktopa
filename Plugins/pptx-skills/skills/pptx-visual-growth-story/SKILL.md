# PPTX Visual Growth Story

Read `pptx-institutional-deck` first, then read the matching orientation skill. Use `pptx-chart-stacked-column` as the chart basis when the visual includes stacked columns.

## Use When

Use this skill for:

- value-build visuals
- AI / ML contribution storytelling
- strategic growth narratives with stacked columns and callouts

This is the closest skill to the stacked-column plus arrow screenshot family.

## Composition Rules

- Use one stacked-column chart as the anchor visual when data supports it.
- Add one directional arrow or trajectory shape only if it reinforces the message.
- Use concise percentage or value callouts, not paragraphs.
- Keep the slide title conclusion-led.

## Placement

- Landscape: anchor the chart in the main chart zone and place callouts above or beside it.
- Portrait: stack the chart first, then keep one arrow and one or two callouts only.

## Formatting Rules

- Keep the arrow restrained and institutional, not illustrative or playful.
- Use the same fixed navy/slate palette as the rest of the deck.
- Add a `Source:` line if the chart data is externally sourced.

## Do Not Use

- decorative arrows with no analytical purpose
- more than one core message on the slide
- dense annotations that should be split into separate slides
