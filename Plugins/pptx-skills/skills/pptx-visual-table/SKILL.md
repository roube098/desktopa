# PPTX Visual Table

Read `pptx-institutional-deck` first, then read the matching orientation skill.

## Use When

Use a table for:

- exposure matrices
- return summaries
- peer comps
- valuation tables
- any dense comparison where exact values matter more than visual ranking

This is the closest skill to the matrix and summary-table screenshots.

## Layout Rules

- Landscape: use the wide body zone for tables with several columns.
- Portrait: keep tables narrow and split them across slides before shrinking text too far.
- Reserve a dedicated bottom source line.

## Formatting Rules

- Header row: `theme.light` fill, `Calibri` semibold, dark text.
- Body: white fill, thin rules, compact row height.
- Use right alignment for numeric columns where possible.
- Keep units in headers or column labels.
- Add `Source:` below the table.

## Do Not Use

- a table when one clear chart tells the story better
- giant unreadable grids
- mixed fonts or decorative cell fills
