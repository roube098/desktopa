# Font families

## Recommended fonts

| Language | Default | Alternatives |
|----------|---------|--------------|
| **English** | **Inter** | Segoe UI, Open Sans, Source Sans 3 |

**Body vs emphasis (product / UI lines):** regular weight in cool charcoal (`363636`–`3D4556`); one emphasized word in **bold** + `000000` (see slide-making-skill tagline example).
